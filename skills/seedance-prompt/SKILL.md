# Seedance 2.0 视频脚本 Prompt 生成器

## 产品背景

本 skill 针对**潮玩手办/毛绒玩偶 IP 角色**视频创作优化。产品特征：
- **毛绒款**: 绒毛质感、针织面料，需强调柔软蓬松感、绒毛纤维细节
- **PVC/搪胶款**: 光滑表面、半透明/珠光材质，需强调光泽反射、精致涂装
- **尺寸**: 微型主体（约10-15cm），拍摄逻辑与真人完全不同
- **IP角色**: 多个固定角色形象，需保持一致性

## 工作流程

### Step 1: 分析输入

读取用户提供的图片，识别：
- **素材类型**: 白底产品图 / 场景图 / 2D插画 / AI生成图
- **角色特征**: 头饰、发型、瞳色花纹、服装材质、标志性配件
- **材质判断**: 毛绒（绒毛/针织）还是硬质（PVC/搪胶/树脂）
- **角色数量**: 单角色 / 双角色 / 多角色群像
- **现有环境**: 白底（需构建场景）/ 已有场景（可沿用）

### Step 2: 确定视频类型

根据素材和需求，判断属于以下哪种场景（如用户已说明则跳过确认）：

| 视频类型 | 典型需求 | 推荐时长 |
|---------|---------|---------|
| **产品展示** | 白底图→360°旋转展示 | 6-10s |
| **玩偶活化** | 让静态玩偶"活过来"，有表情动作 | 8-15s |
| **场景故事** | 玩偶在特定场景中演绎小剧情 | 10-15s |
| **多角色互动** | 2-4个角色同框互动 | 10-15s |
| **节日主题** | 春节/圣诞/万圣节等节日内容 | 8-12s |
| **插画动态化** | 2D插画→动态视频 | 6-10s |
| **氛围混剪** | 多个意境画面拼接，无剧情关联 | 10-15s |

同时确认：
1. **情绪基调**: 可爱治愈 / 温馨日常 / 神秘奇幻 / 搞怪有趣 / 史诗感 / 暗黑哥特
2. **音频需求**: 环境音效 / 轻柔配乐 / 八音盒风格 / 无音乐

### Step 3: 影视理论决策

#### 微型主体的景别体系

玩偶拍摄的景别与真人不同，"远景"是桌面尺度，"特写"是微距尺度：

| 景别 | Seedance关键词 | 在玩偶拍摄中的含义 |
|------|---------------|-----------------|
| 远景 | 全景俯拍、微缩世界全景 | 展示玩偶所在的完整场景（桌面/微缩世界） |
| 全景 | 全身镜头、玩偶全身 | 玩偶完整身体+少量环境 |
| 中景 | 中景、半身 | 玩偶上半身，可看到手部动作 |
| 近景 | 近景、面部 | 玩偶面部+头饰细节 |
| 微距特写 | 微距特写、极近距离 | 眼睛瞳孔花纹、绒毛纤维、涂装细节 |

#### 运镜选择

| 运镜 | Seedance关键词 | 玩偶场景最佳用法 |
|------|---------------|---------------|
| 环绕 | 环绕运镜、围绕旋转 | 产品展示360°、角色出场仪式感 |
| 推近 | 镜头缓缓推近 | 从全景推到面部，揭示表情细节 |
| 拉远 | 镜头缓缓拉远 | 揭示微缩世界全貌 |
| 低角度仰拍 | 低角度仰视、蚂蚁视角 | 让小玩偶显得高大，代入微缩世界 |
| 俯拍 | 俯视镜头、上帝视角 | 展示场景布局、多角色关系 |
| 微距跟焦 | 微距、浅景深、焦点转移 | 突出材质细节（绒毛/光泽） |
| 滑轨横移 | 横向滑动、水平移动 | 多角色并排展示、场景巡览 |
| 固定 | 固定镜头 | 玩偶在画面中做动作，凝视感 |

#### 材质专用光影描述

**毛绒玩偶**:
- 「柔和的漫射光，温暖的光线照在绒毛表面，每根纤维清晰可见」
- 「逆光照射，绒毛边缘发出温暖的光晕」
- 「暖色调柔光，营造出柔软蓬松的触感」

**PVC/搪胶手办**:
- 「侧光打亮，表面呈现细腻的珠光反射」
- 「光线穿过半透明材质，内部微微发光」
- 「工作室三点布光，产品表面高光与阴影分明」

#### 声音设计（玩偶场景适配）

- 「轻柔的八音盒旋律 + 细微的环境音效」→ 可爱治愈
- 「仅生成环境音效，不要配背景音乐」→ ASMR质感，突出材质声
- 「欢快活泼的配乐」→ 搞怪/节日
- 「神秘空灵的背景音」→ 奇幻/暗黑主题

### Step 4: 组装 Prompt

#### 素材引用语法

```
@图片1 作为角色参考 / 作为首帧
@图片2 作为第二个角色 / 作为场景参考
@视频1 参考镜头语言 / 参考动作
@音频1 用于配乐 / 参考节奏
```

#### Prompt 结构模板

**单角色产品展示/活化**:
```
@图片1 作为角色参考。[材质风格描述]的玩偶角色，[场景环境]。[音频指令]。

第一个画面：[景别] + [运镜] + [初始状态/动作] + [光影]
第二个画面：[景别变化] + [运镜] + [动作发展] + [细节]
```

**多角色互动**:
```
@图片1 和 @图片2 是两个玩偶角色。[场景描述]，两个角色[互动关系]。[音频指令]。

第一个画面：[建立镜头，展示场景和角色关系]
第二个画面：[聚焦互动动作]
第三个画面：[情感高潮/收束]
```

**插画动态化**:
```
@图片1 作为首帧。保持画面中的插画风格和色彩，让画面中的角色产生[具体动态]，[环境元素动态]。[音频指令]
```

### Step 5: 输出

1. **创作分析**（2-3句）: 基于素材类型和材质，解释选择的影视语言
2. **完整 Prompt**: 可直接复制粘贴到即梦
3. **参数建议**: 推荐时长、宽高比、是否需要额外素材

## Prompt 输出规范

### 必须遵守
- Prompt 用**中文**书写
- 每个分镜必须包含：**景别 + 运镜 + 动作 + 氛围**
- 分镜之间要有**节奏变化**（景别递进、运镜对比、动静交替）
- 音频指令放在分镜之前
- 描述玩偶动作时用**拟人化但符合体型的动作**（歪头、眨眼、小手挥动、蹦跳）
- 白底产品图必须在 prompt 中**构建场景环境**，不能让玩偶悬浮在白色空间

### 玩偶动作词库
- 头部: 歪头、缓缓转头、抬头望向天空、低头沉思
- 眼睛: 眨眼、眼睛闪烁光芒、目光注视镜头
- 身体: 微微摇晃、蹦跳、站起、坐下、转身
- 手部: 小手挥动、举起道具、双手捧住物品、招手
- 互动: 相互对视、拉手、拥抱、递东西、一起奔跑
- 情绪: 开心地蹦跳、害羞地捂脸、好奇地探头、安静地坐着

### 微缩世界场景词库
- 自然: 苔藓地面、蘑菇丛林、花瓣铺成的小路、露珠反射光芒
- 室内: 微缩书房、小茶桌、迷你壁炉、缩小的窗台
- 节日: 迷你圣诞树、小灯笼、微型烟花、缩小版年夜饭桌
- 奇幻: 水晶洞穴、星空下的草地、月光湖面、飘浮的萤火虫
- 都市: 微型咖啡馆、小路灯、迷你长椅、缩小的街道

### 避免
- 不要写"生成一段视频"之类的元指令
- 不要堆砌过多运镜（每个画面一种主运镜）
- 不要写超过 5 个分镜
- 避免写实真人脸部描述（平台限制）
- 不要让玩偶做**不符合体型比例的大幅动作**（如奔跑跳远、格斗翻滚）

## 参考示例

### 示例 1: 毛绒玩偶活化（白底图→场景故事）

**Prompt**:
```

第三个画面：低角度仰拍，一片金黄落叶飘落到玩偶的苹果帽上，玩偶抬头望向天空，暖色调柔光。
```

## 多模态组合策略

| 素材组合 | 引用方式 | 适用场景 |
|---------|---------|---------|
| 1张白底图 + 文字 | @图片1 作为角色参考 + prompt构建场景 | 让玩偶"活"在新场景里 |
| 1张白底图 + 文字 | @图片1 作为首帧 | 保持白底的产品展示旋转 |
| 2张角色图 + 文字 | @图片1/@图片2 作为两个角色 | 双角色互动故事 |
| 多张角色图 + 文字 | 每张用@图片N指定角色 | 群像场景（建议≤4角色） |
| 产品图 + 场景参考图 | @图片1 角色 + @图片2 场景/风格参考 | 指定特定场景氛围 |
| 产品图 + 参考视频 | @图片1 角色 + @视频1 参考运镜 | 复刻特定运镜效果 |
| 插画 + 文字 | @图片1 作为首帧 | 2D插画动态化 |
| 产品图 + 音频 | @图片1 角色 + @音频1 配乐参考 | 音乐卡点视频 |
